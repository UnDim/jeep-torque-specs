# Jeep Torque Specs
This application allows you to search torque specs based on model (TJ, JK, JL, JT) and Category (Body, Brakes, etc).

## Running Locally

```sh
$ heroku local web
```

Your app should now be running on [localhost:5000](http://localhost:5000/).

## Deploying to Heroku

```
$ heroku create
$ git push heroku main
$ heroku open
```
